# JewishMusic-Previewer.org
üéµ Jewish Music Preview Ultra
Jewish Music Preview Ultra is a high-performance, single-page web application designed to streamline the discovery of Jewish music. By utilizing 30-second high-quality previews, the platform allows users to quickly "capture the vibe" of a track without having to listen to full songs or manually skip through irrelevant content.

üéØ Our Goal
The primary objective of this project is to eliminate "search fatigue." Instead of spending time skipping through full tracks to find something you enjoy, this app presents the most essential part of the song immediately. This makes it easier to curate your personal library and discover new artists in a fraction of the time.

‚ú® Key Features
Dual-Layer Filtering Engine: Features an Admin Dashboard to manage Whitelists (approved artists) and Blacklists (blocked keywords) to maintain a focused experience.

Dynamic UI Scaling: Includes a unique "App Scale" system to optimize the layout for everything from small monitors to large-scale TV/Cinema displays.

Audio Visualization: A real-time frequency analyzer provides a visual representation of the music during playback.

Theater Mode: Dim the entire interface to focus purely on the album art and the music.

Sleep Timer: Built-in countdown functionality to automatically stop music after a set duration.

üöÄ Getting Started
Initial Setup: When you first launch the app, you will be prompted to select a Startup Scale. Choose the one that best fits your screen size.

Accessing Admin Features: * Click the Settings icon.

Click the Admin button.

Enter the default PIN (found in the source code under admin.pin) to access Whitelist, Blacklist, and User Report management.

Building your Library: Use the search bar to find artists. Click the "Star" icon to save songs to your local library for quick access later.
